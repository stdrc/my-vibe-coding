# My Vibe Coding

A collection of my vibe coding projects.

- [chinese-colors](https://github.com/stdrc/chinese-colors): A static website for browsing Chinese traditional colors with search/filter and quick HEX copy.
- [macos-init-trajectory-viz](https://github.com/stdrc/macos-init-trajectory-viz): A trajectory visualizer for macOS initialization/configuration conversations in Codex sessions.
- [hakimi](https://github.com/stdrc/hakimi): A CLI bot bridge that lets you chat via Telegram/Slack/Feishu to remotely control your computer through an AI assistant.
- [tldr-vscode](https://github.com/stdrc/tldr-vscode): A VS Code extension that shows code structure (like stubs) and structure diffs without implementation noise.
- [longshot](https://github.com/stdrc/longshot): A smart screenshot stitching tool that auto-detects overlaps and merges long screenshots.
- [locklock](https://github.com/stdrc/locklock): A Next.js resource allocation and management system with auth and real-time status updates.
- [reading-list](https://github.com/stdrc/reading-list): A simple web app for managing and displaying a personal reading list.
- [music-collection](https://github.com/stdrc/music-collection): Scripts plus a static site for collecting and presenting album/playlists data from music services.
